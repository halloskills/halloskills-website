# HalloSkills Website — Team Onboarding
### Änderungen über Claude Code vornehmen

---

## Was du brauchst (einmalig einrichten)

**1. Claude Code installieren**
- Geh auf **claude.ai/download** und lade Claude Code herunter
- Mit dem HalloSkills Team-Account einloggen

**2. GitHub Desktop installieren & anmelden**
- Geh auf **desktop.github.com** und lade GitHub Desktop herunter
- Nach der Installation: GitHub Desktop öffnen
- Oben links auf **„Sign in to GitHub.com"** klicken
- Mit dem HalloSkills GitHub-Account einloggen: **admin@halloskills.de**
- ✅ Einmalig — danach nie wieder nötig

**3. Repository klonen**
- In GitHub Desktop: **File → Clone Repository**
- URL eingeben: `github.com/halloskills/halloskills-website`
- Speicherort wählen und auf **Clone** klicken

**4. Kontext eingeben (nur beim ersten Mal)**
- Claude Code öffnen und schreiben:
  > *„Das ist die HalloSkills Website. Alle Inhalte sind in `index.html`, Bilder in `Assets/Bilder/`, Team-Fotos in `Assets/Bilder/Team/`, Icons in `Assets/Website-icons/`. Die Website läuft auf GitHub Pages."*

---

## Änderungen vornehmen

**Schritt 1 — In Claude Code beschreiben was geändert werden soll:**

| Was du willst | Was du schreibst |
|---|---|
| Bild ersetzen | *„Ersetze das Hero-Bild durch `neues-bild.png` aus `Assets/Bilder/`"* |
| Text ändern | *„Ändere den Kursnamen von 'Kaufmann E-Commerce' auf 'E-Commerce Kaufmann'"* |
| Team-Foto | *„Ersetze das Foto von Heiko durch `Heiko-neu.png` im Team-Ordner"* |
| Neuer Kurs | *„Füge einen neuen Kurs 'IT-Systemkaufmann' hinzu, gleich aufgebaut wie die anderen"* |

Claude Code ändert die Datei automatisch und speichert sie.

**Schritt 2 — In GitHub Desktop pushen:**
1. GitHub Desktop öffnen → die Änderung erscheint automatisch links
2. Unten links: kurze Beschreibung eingeben (z. B. *„Foto von Heiko aktualisiert"*)
3. Auf **„Commit to main"** klicken
4. Oben auf **„Push origin"** klicken
5. ✅ Die Website ist innerhalb ~1 Minute live

---

## Neues Bild hinzufügen

1. Bild in den richtigen Ordner kopieren:
   - Team-Fotos → `Assets/Bilder/Team/`
   - Andere Bilder → `Assets/Bilder/`
2. In Claude Code beschreiben welches Bild wo eingesetzt werden soll
3. Dann wie oben: GitHub Desktop → Commit → Push
