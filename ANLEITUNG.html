Team-Onboarding

Änderungen über Claude Code vornehmen

Was du brauchst (einmalig einrichten)

1
Claude Code installieren

Geh auf claude.ai/download und lade Claude Code herunter. Mit dem HalloSkills Team-Account einloggen.

2
GitHub Desktop installieren & anmelden

Geh auf desktop.github.com und lade GitHub Desktop herunter. Nach der Installation öffnen → „Sign in to GitHub.com" klicken → mit dem HalloSkills GitHub-Account einloggen: admin@halloskills.de.

✅ Einmalig — danach nie wieder nötig
3
Repository klonen

In GitHub Desktop: File → Clone Repository → URL eingeben: github.com/halloskills/halloskills-website → Speicherort wählen → Clone klicken.

4
Kontext eingeben (nur beim ersten Mal)

Claude Code öffnen und folgendes schreiben:

„Das ist die HalloSkills Website. Alle Inhalte sind in index.html, Bilder in Assets/Bilder/, Team-Fotos in Assets/Bilder/Team/, Icons in Assets/Website-icons/. Die Website läuft auf GitHub Pages."

Änderungen vornehmen

Schritt 1 — In Claude Code beschreiben was geändert werden soll:

Was du willst Was du schreibst
Bild ersetzen „Ersetze das Hero-Bild durch neues-bild.png aus Assets/Bilder/"
Text ändern „Ändere den Kursnamen von 'Kaufmann E-Commerce' auf 'E-Commerce Kaufmann'"
Team-Foto „Ersetze das Foto von Heiko durch Heiko-neu.png im Team-Ordner"
Neuer Kurs „Füge einen neuen Kurs 'IT-Systemkaufmann' hinzu, gleich aufgebaut wie die anderen"

Claude Code ändert die Datei automatisch und speichert sie.

Schritt 2 — In GitHub Desktop pushen:

  • 1GitHub Desktop öffnen → die Änderung erscheint automatisch links
  • 2Unten links: kurze Beschreibung eingeben (z. B. „Foto von Heiko aktualisiert")
  • 3Auf „Commit to main" klicken
  • 4Oben auf „Push origin" klicken
  • ✅Die Website ist innerhalb ~1 Minute live

Neues Bild hinzufügen

  • 1Bild in den richtigen Ordner kopieren: Team-Fotos → Assets/Bilder/Team/  ·  Kursbilder → Assets/Bilder/Umschulungen/  ·  sonstige Bilder → passenden Unterordner in Assets/Bilder/
  • 2In Claude Code beschreiben welches Bild wo eingesetzt werden soll
  • 3Dann wie oben: GitHub Desktop → Commit → Push

Do's & Don'ts

✅ Das kannst du anfragen
  • Texte und Überschriften ändern
  • Bilder ersetzen
  • Team-Mitglieder hinzufügen oder ändern
  • Kurse bearbeiten
  • Kleine Designanpassungen (Farbe, Abstände)
❌ Bitte nicht
  • Komplett neue Sektionen oder Unterseiten → zuerst mit Nina abstimmen
  • Änderungen die das grundlegende Layout betreffen
